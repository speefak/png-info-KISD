# png-info-KISD
shown png info, added ( Caption ) or created by stable diffusion ( parameter )

	  -h			=> help dialog \n"
	  -si			=> show script information \n"
	  -cfrp			=> check for required packets \n"
	  -g			=> (g) GUI info (zenity)\n"



useable with nemo (tested) nautilus (untested)
